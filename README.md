# Description
> This project is with Serverless Framework and NestJs, Serverless is with Lambdas functions and with lambda layers, while NestJs is without HTTP. It includes SSM. V3

## AWS

- Config aws credentials
- Set stack name, profile and domain in ServerlessScripts.js
- module.exports.stackName = () => 'serverless-nestjs-ssm';
- module.exports.profile = () => `profile_${this.stage()}`;
- module.exports.domainName = () => `api-${this.stage()}.domain.com`;

## Installation
```sh
$ npm install
```
## Test
```sh
$ npm run validate
```
## Deploy
```sh
$ npm run deploy
```
